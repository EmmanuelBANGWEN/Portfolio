Feravi Stat : Application de suivi climatique et de production d'œufs pour les fermes avicoles

Feravi Stat est une application web interactive pour aider les éleveurs de poules pondeuses à mieux comprendre l’impact des conditions climatiques sur la production d’œufs. L’objectif est de faciliter la prise de décision grâce à une meilleure lecture des données environnementales et de production
